## Description
アライさんマンションのゲームです。  
Javardryにて制作されています。
JAVAランタイムが動くPCにて動作します。  

某掲示板の手書きデータをJavardryのイベントとして作る試みです。  

データを弄る場合は、JavardryEditerでシナリオデータを読み込んで、  
データファイルgameData.datを出力してください。

その後、下記のDemo配布サイトで配布しているファイルの中でdata/gameData.datを上書きして下さい。
